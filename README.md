# MOMR-Manager

#### 介绍
微型微生物在线培养反应器的上位机代码，考虑到简洁性，使用最基础的python库。方便安装调试。
本程序的设计初衷是将python脚本结合到微生物培养控制过程中。
首先先配合我自己开发的MOMR微型在线微生物培养反应器来使用，逐步实现功能，将来只要修改momr模型中的
数据解析和打包功能就可以扩展给其他类型的反应器使用。
